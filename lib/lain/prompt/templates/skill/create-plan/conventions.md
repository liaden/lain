## Conventions this plan must honour

Defer to the repo's CLAUDE.md and the approved design plan it names for the binding rules —
toolchain, output discipline, the require manifest, the commit-grouping traps. This hole is
where a project pins the *planning* conventions the shipped scaffold cannot know: the planning
doc location and slug style, the review-panel roster, the default `commit-mode`, and any
house rule a card author must not violate. Override it at
`.lain/slots/skill/create-plan/conventions.md`.
